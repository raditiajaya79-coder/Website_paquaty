# Pakuaty Website — React + Vite + Tailwind CSS v4

Project website company profiling dan katalog produk menggunakan stack modern.

## Stack
- **Frontend Framework**: React 19
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS v4 (Native Vite integration)
- **Routing**: React Router v7
- **SEO**: React Helmet Async

## Struktur Folder
- `src/pages`: Daftar halaman (Home, About, Products, Contact, dll)
- `src/components`: Komponen UI reusable (Navbar, Footer, dll)
- `src/layouts`: Wrapper layout utama
- `src/utils`: Helper (SEO, formatting, dll)
- `src/assets`: Aset statis (gambar, icon, dll)

## Cara Menjalankan
1. Masuk ke folder: `cd folder_website`
2. Install dependencies: `npm install`
3. Jalankan development server: `npm run dev`
4. Buka `http://localhost:5173` di browser
